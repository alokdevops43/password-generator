# 🔐 Password Generator

A modern and beginner-friendly Password Generator built using **HTML**, **CSS**, and **Vanilla JavaScript**.

This project allows users to generate secure passwords by selecting different character types and password lengths. It also includes a password strength indicator and a copy-to-clipboard feature.
##live link

https://alokdevops43.github.io/password-generator/


---

## 🚀 Features

- Generate random passwords
- Adjustable password length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include symbols
- Password strength indicator
- Copy password to clipboard
- Responsive design
- Clean and modern UI
